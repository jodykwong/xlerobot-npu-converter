**问题描述**:

**具体表现**:
- `src/npu_converter/core/` 目录完全不存在
- PTQ转换器实现了应属于core的基础功能
- 违反了分层架构原则

**代码重复问题**:
- PTQ转换器中的ModelInfo类应属于core/models/
- ValidationResult类应属于core/models/
- 基础模型加载逻辑应属于core/converter.py
- 错误处理机制应属于core/utils/
